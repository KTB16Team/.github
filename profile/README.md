## 기여 방법
이 프로젝트에 기여하려면 다음 단계를 따라주세요:

1. 이슈를 확인하고 작업할 항목을 선택하세요.
2. 새로운 브랜치 생성:
   ```bash
   git checkout -b feature/새로운기능
   ```
3. 작업 후 커밋:
   ```bash
   git commit -m "추가: 새로운 기능"
   ```
4. Pull Request(PR) 생성:
   ```bash
   git push origin feature/새로운기능
   ```
